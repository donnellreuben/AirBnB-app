1. Check Ruby Version
Edit .ruby-version to the version of Ruby you want to use then 

2. Bundle Install
bundle install

3. Database
rails db:setup 

4. Server, either
rails server 
or 
bin/dev *recommended*

5. To view AirBnb
localhost:3000
